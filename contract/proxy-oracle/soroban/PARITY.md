# NEAR/Soroban Proxy Oracle Parity

This document pins the behavioral parity baseline between the NEAR and Soroban implementations of the Templar Proxy Oracle.

**Baseline Hash**: `64bf8b821cabbc94e4591ca89997c8ec00f365c7`
